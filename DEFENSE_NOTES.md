# Defense Notes

## Главная идея проекта
Проект представляет собой сервис онлайн-заказа и доставки товаров с полноценной frontend и backend частями.

## Что реализовано
- Регистрация и авторизация пользователей
- JWT-аутентификация
- Каталог товаров
- Страница товара
- Корзина
- Оформление заказа
- История заказов
- Админ-панель для управления товарами
- Админ-панель для управления заказами
- Загрузка изображений товаров
- Защищенные маршруты для админа
- Пустые состояния, загрузки, 404

## Технологии
- Frontend: React, React Router DOM, Context API, CSS
- Backend: Node.js, Express, MongoDB, Mongoose, JWT, bcryptjs, multer

## Ключевые преимущества
- Разделение frontend и backend
- Работа с реальной базой данных MongoDB
- Защита routes и роли пользователя
- Работа с загрузкой файлов
- Хранение состояния корзины в localStorage
- Полный пользовательский сценарий: выбор товара, корзина, заказ

## Что показать на защите
- Как работает регистрация и логин
- Как добавлять товары в корзину и оформлять заказ
- Как админ создает/редактирует/удаляет товары
- Как админ меняет статус заказа
- Примеры пустых состояний и загрузок
- Как проект работает с изображениями и хранит их на сервере

## Важные замечания
- Для доступа в админ-панель пользователь должен иметь роль `admin`
- После изменения роли в базе нужно повторно залогиниться
- Файлы загружаются в папку `server/uploads`
- API доступно по `http://localhost:5000/api`
