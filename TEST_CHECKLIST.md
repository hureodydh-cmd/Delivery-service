# Test Checklist

- [ ] Вход в систему
- [ ] Просмотр каталога товаров
- [ ] Открытие страницы товара
- [ ] Добавление товара в корзину
- [ ] Просмотр корзины
- [ ] Оформление заказа
- [ ] Просмотр истории заказов

- [ ] Открытие страницы управления заказами
- [ ] Изменение статуса заказа
