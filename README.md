# Сервис онлайн-заказа и доставки товаров

## Автор
Алимхан

## Описание проекта
«Сервис онлайн-заказа и доставки товаров» — это fullstack веб-приложение, разработанное с использованием React, Node.js и MongoDB.

Пользователь может:
- зарегистрироваться и войти в систему;
- просматривать каталог товаров;
- открывать страницу отдельного товара;
- добавлять товары в корзину;
- оформлять заказ;
- просматривать историю своих заказов.

Администратор может:
- добавлять новые товары;
- редактировать товары;
- удалять товары;
- загружать изображения товаров;
- просматривать все заказы;
- изменять статус заказов.

## Установка и запуск проекта

1. Установка зависимостей frontend
```bash
cd my-react-app
npm install
```
2. Установка зависимостей backend
```bash
cd ../server
npm install
```

## Настройка переменных окружения

Создайте файл .env в папке server:

```bash
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/online_store_db
JWT_SECRET=my_super_secret_key
```

## Запуск проекта

### Запуск backend
```bash
cd server
npm run dev
```

### Запуск frontend
```bash
cd my-react-app
npm run dev
```
